# Procedural Generation

Stuck on an idea? Use the Procedural Generators to create content instantly.

## Available Generators

Access these from the **Tools** menu:

- **Bestiary Generator**: Creates unique monsters with descriptions, stats, and behaviors.
- **Location Generator**: Generates dungeons, cities, or wildnerness locations with descriptions and points of interest.
- **Item & Culture Generator**: Creates magic items, artifacts, or details about cultures and factions.

## Using Generators

1. Open the desired generator tool.
2. Select a context or theme (e.g., "Swamp", "Ancient Ruins", "High Magic").
3. Click **Generate**.
4. Review the output. You can copy the text or save it directly to a new file in your project.
