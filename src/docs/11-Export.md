# Exporting

When your content is ready, you can export it for publication.

1. Go to **File > Export...** (or click the Export button in the menu).
2. Select the output format (e.g., PDF, HTML, JSON).
3. Configure export options (layout, styling, included files).
4. Click **Export**.

CodexLotus compiles your markdown files, processes stat blocks and images, and generates the final document.
