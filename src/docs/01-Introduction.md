# Welcome to CodexLotus

CodexLotus is a specialized integrated development environment (IDE) for Tabletop RPG (TTRPG) creators. It combines powerful text editing with specialized tools for world-building, rule design, and content generation.

## Key Features

- **Project Management**: Organize your RPG content into projects.
- **Markdown Editor**: Write your content with a specialized editor that supports stat blocks and live previews.
- **Stat Block Designer**: Create and manage monster and item stat blocks visually.
- **Lore Map**: Visualize the relationships between characters, locations, and events in your world.
- **Dice Tools**: Calculate probabilities and roll dice directly in the app.
- **Rule Calculators**: Balance your game with calculators for XP, economy, and encounter difficulty.
- **Procedural Generation**: Generate creatures, locations, and items to flesh out your world.
- **Playtest Simulator**: Run scenarios to test your rules and balance.
- **Auto Tagger**: Automatically tag and link concepts across your documentation.
- **Export**: Export your project to PDF or other formats for publishing.

## Getting Started

To get started, create a new project or open an existing one from the **File** menu. Use the sidebar to navigate your files and the **Tools** menu to access the various utilities.
