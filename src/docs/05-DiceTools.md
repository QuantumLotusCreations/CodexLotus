# Dice Tools

CodexLotus includes tools for rolling dice and analyzing probability, essential for balancing game mechanics.

## Dice Probability Calculator

1. Go to **Tools > Dice Probability Calculator**.
2. Enter a dice formula (e.g., `2d6 + 3`, `1d20`, `4d6kh3`).
3. The tool visualizes the probability distribution graph, showing the likelihood of rolling each number.
4. View statistics like Mean, Minimum, and Maximum values.

This helps you understand the mathematical impact of the rules you are designing.
