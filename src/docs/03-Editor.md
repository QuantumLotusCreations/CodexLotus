# Markdown Editor

The core of CodexLotus is a powerful Markdown editor designed for RPG writing.

## Basic Editing

Click on any file in the Explorer to open it in the editor. The editor supports standard Markdown syntax:

- **Bold**: `**text**`
- *Italic*: `*text*`
- # Headings: `# Heading 1`, `## Heading 2`, etc.
- Lists: `- Item` or `1. Item`
- Links: `[Text](url)`

4. Review the changes in the Diff Viewer.

## Stat Block Insertion

You can easily insert stat blocks into your text:

1. Click the **Stat Block** button in the layout toolbar (or use the Tools menu).
2. Select a template or enter the stat block details.
3. The stat block code will be inserted into your document.
